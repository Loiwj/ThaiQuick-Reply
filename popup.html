Xuất & nhập reply bình luận Thái

📋 Hướng dẫn Reply JSON
Bước 1: Mở YouTube Studio → Comments
Bước 2: Nhấn nút 💬 ở góc phải dưới trang
Bước 3: Click "📋 Quét & Export JSON" → Copy
Bước 4: Paste JSON vào ChatGPT/Gemini để AI trả lời
Bước 5: Copy JSON phản hồi → paste vào ô "Import"
Bước 6: Nhấn "👁️ Xem trước" → kiểm tra → "✅ Xác nhận"
Bước 7: Chỉ cần nhấn "Phản hồi" cho từng comment!
💡 Format JSON reply từ AI:
{
  "replies": [
    {
      "id": 1,
      "author": "@sponsor9091",
      "comment": "ต่อเลยครับ",
      "translation": "Tiếp tục đi",
      "reply": "ขอบคุณครับ จะทำต่อนะครับ 🙏",
      "replyTranslation": "Cảm ơn, sẽ làm tiếp nhé 🙏"
    }
  ]
}
